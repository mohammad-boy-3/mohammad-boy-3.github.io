ژنراتور عدد تصادفی با جلوه‌های بصری جادویی

✨ معرفی برنامه

ژنراتور عدد تصادفی یک برنامه جذاب و کاربردی است که تجربه‌ای متفاوت از تولید اعداد تصادفی را برای شما فراهم می‌کند. با استفاده از جلوه‌های بصری خیره‌کننده و انیمیشن‌های پیشرفته، هر بار که عددی تولید می‌کنید، یک تجربه بصری منحصر به فرد خواهید داشت.

این برنامه با الهام از هنر و معماری ایرانی طراحی شده و فضایی مرموز و جادویی را برای کاربران ایجاد می‌کند. اعداد به صورت زنده و با انیمیشن‌های خاص نمایش داده می‌شوند و سیستم ذرات سه‌بعدی فضایی جادویی را به وجود می‌آورد.

🎯 ویژگی‌های برنامه

🎲 تولید اعداد تصادفی

تولید اعداد تصادفی در بازه دلخواه از 1 تا 100,000,000,000 با الگوریتم پیشرفته

✨ جلوه‌های بصری

انیمیشن‌های خیره‌کننده، سیستم ذرات سه‌بعدی و افکت‌های نورپردازی پیشرفته

📊 آمار و تحلیل

نمایش آمار کامل، تاریخچه اعداد، تحلیل فراوانی و نمودارهای تعاملی

🍀 اعداد شانس

حالت خاص برای تولید اعداد شانس با افکت‌های ویژه و پیش‌بینی‌های جالب

⚙️ فناوری‌های استفاده شده

HTML5 CSS3 JavaScript ES6+ Anime.js Tailwind CSS Canvas API Web Audio API

این برنامه با استفاده از جدیدترین فناوری‌های وب ساخته شده و بهینه‌سازی کامل برای دستگاه‌های موبایل دارد. سیستم ذرات با استفاده از Canvas API پیاده‌سازی شده و انیمیشن‌ها با کتابخانه Anime.js اجرا می‌شوند.

📖 نحوه استفاده

1. بازه مورد نظر برای تولید اعداد را تنظیم کنید (حداقل و حداکثر)
2. دکمه "تولید عدد تصادفی" را فشار دهید و از تماشای انیمیشن لذت ببرید
3. آمار و تاریخچه اعداد را در صفحه آمار مشاهده کنید
4. برای تجربه‌ای متفاوت، به صفحه "عدد شانس" بروید
5. تنظیمات مورد نظر خود را در هر صفحه تنظیم کنید
